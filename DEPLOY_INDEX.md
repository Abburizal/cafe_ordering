## ğŸ› Common Issues & Solutions

### Issue: "Git not found"
**Solution:** Install Git from https://git-scm.com/downloads

### Issue: "No GitHub remote"
**Solution:** 
```bash
git remote add origin https://github.com/YOUR_USERNAME/cafe-ordering.git
```

### Issue: "Permission denied" (Bash script)
**Solution:**
```bash
chmod +x deploy-render.sh
./deploy-render.sh
```

### Issue: "Database connection error"
**Solution:**
1. Check environment variables in Render
2. Verify database credentials
3. Ensure database is imported

### Issue: "Website shows 500 error"
**Solution:**
1. Check Render logs
2. Verify config.php exists
3. Check database connection
